# Parking Management System

A Python-based parking management system with QR code support.  
This project allows you to manage parking spaces efficiently and generate QR codes for tickets or vehicle tracking.

---

## Features

- Manage parking slots and occupancy
- Generate QR codes for tickets or vehicle identification
- Simple and intuitive Python-based CLI
- Uses OpenCV for optional image/video processing

---

## Requirements

- Python 3.13+
- Virtual environment tool (`uv` recommended)
